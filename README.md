📘 PDF ぱっか 1.0

見開き対訳PDFを、シンプルに作るツール

「左ページPDF」と「右ページPDF」を選ぶだけで、
**左右交互に並んだ“見開き対訳PDF”**を自動生成します。

研究資料・翻訳・対訳読書・TRPG資料などに最適です。

✨ 特徴

📄 左右PDFをページ単位で交互結合

🗂️ 複数PDF対応（ファイル名順で自動整列）

🪞 表紙PDF（mirror_book_cover.pdf）を自動で先頭追加

🖥️ PDF閲覧用 / 印刷用 出力モード切替

💾 保存先フォルダを明示指定

🌐 多言語UI対応（Language CSV）

🧩 UIとロジックを完全分離した設計

🖱️ 使い方

言語を選択

左側ページPDF を選択

右側ページPDF を選択

出力モードを選択

PDF閲覧用（デフォルト）

印刷用

保存先フォルダ を指定

スタート を押す

→ 見開き対訳PDFが生成され、指定フォルダに保存されます。

📌 制限事項・注意

左右PDFのページ数が異なる場合
→ 少ない方のページ数に合わせて生成されます

ブラウザの File System Access API を使用
→ 一部ブラウザ／iOSでは制限があります

TRPG資料の原書＋翻訳

学習用見開き教材
